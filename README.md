#  Todo List using DOM
A simple and interactive Todo List project build using **HTML,CSS and Javascript(DOM Manipulation)**.
Users can add,delete,and mark tasks as completed

## 🚀 Live Demo
👉• Live: https://todo-list-using-dom.vercel.app 


#  Features
Add new todos
Mark todos as completed
Delete todos
Data stored in browser(using localStorage)

# Tech Stack
- HTML
- CSS
- JavaScript(DOM)
